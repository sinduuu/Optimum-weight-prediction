# optimum-weight-prediction

The objective of this project is to predict the weight of products of a FMCG company for various warehouses with different conditions, size & locality. To determine the Ideal Quantity of Product Weight Shipped to the various warehouses of FMCG Instant Noodles Company in order to reduce wastage of the Product, Bridge the Demand â€“ Supply Gap and avoid over-stacking of Products in the warehouses.

Models used:

a. Linear Regression

b. Ridge Regression

c. Lasso Regression

d. Decision Tree Regression

e. Random Forest Regression 

f. Gradient Boosting Regression 

Best model: Gradient boosting with an accuracy of 99.4%
